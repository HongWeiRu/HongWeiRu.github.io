# 零售貨架管理系統 - ShelfManager Pro

## 📋 專案簡介

ShelfManager Pro 是一個功能完整的零售貨架管理系統，提供現代化的響應式網頁界面，支援桌面、平板和手機設備。系統包含貨架管理、庫存控制、銷售分析、客戶管理等企業級功能。

## 🚀 主要功能

### 📊 核心管理功能
- **總覽面板** - 營運數據即時監控
- **貨架管理** - 完整的CRUD操作與拖拉編輯
- **商品庫存** - 庫存追蹤與補貨提醒
- **明星商品** - 熱銷商品分析與排行

### 📈 分析報表系統
- **銷售報表** - 多維度銷售數據分析
- **客戶分析** - 客戶分群與行為分析
- **獲利分析** - 成本結構與獲利優化

### ⚙️ 系統管理
- **系統設定** - 用戶管理、通知設定、備份還原
- **資料同步** - 多系統整合與同步監控

## 🛠️ 快速開始

### 1. 下載檔案
請下載以下檔案到同一個資料夾：
- `index.html`
- `styles.css`
- `script.js`

### 2. 開啟系統
使用瀏覽器開啟 `index.html` 檔案即可開始使用。

### 3. 系統需求
- **瀏覽器支援**：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **響應式支援**：桌面 (1200px+), 平板 (768px-1024px), 手機 (<=768px)
- **PDA裝置**：特別優化 480px 以下螢幕

## 🔧 自定義配置

### 修改配色
在 `styles.css` 中調整 CSS 變數：
```css
:root {
    --primary-color: #10b981;
    --secondary-color: #3b82f6;
    --accent-color: #f59e0b;
}
```

### 調整佈局
修改響應式斷點：
```css
@media (max-width: 768px) {
    /* 手機版樣式 */
}
```

### 擴展功能
在 `script.js` 中添加新功能：
```javascript
function newFeature() {
    // 新功能實作
}
```

## 🔌 API 整合

系統預留了 API 整合介面：

```javascript
// 同步到後台範例
function syncToBackend(data) {
    fetch('/api/shelves', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(data)
    });
}
```

## 📞 技術支援

### 常見問題

**Q: 為什麼拖拉功能不工作？**
A: 請確保點擊「編輯」按鈕進入編輯模式。

**Q: 手機版選單如何開啟？**
A: 點擊右上角的 ☰ 按鈕。

**Q: 如何匯出報表？**
A: 在各功能頁面點擊「匯出報表」按鈕。
